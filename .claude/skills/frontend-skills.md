# Frontend Skills

Use this skill pack when building the frontend.

## Skill Areas

- reference decomposition
  - extract layout zones, repeated card patterns, filters, and modal content from `frontend/index.html`
- App Router planning
  - split landing, auth, and app experiences into route groups
- MUI theming
  - define light theme tokens, typography, spacing, cards, pills, and charts consistently
- state design
  - separate auth, guest session, model catalog, active model, chat, and marketplace filter state
- browser API integration
  - wrap Web Speech API, `SpeechSynthesis`, and camera snapshot flows safely in client components
- large-list UX
  - use pagination or virtualization for 400+ models and marketplace cards
- capability-aware chat UX
  - keep unsupported image/audio/video/vision actions useful via facilitation prompts
- responsive adaptation
  - preserve reference hierarchy while collapsing side panels intentionally on smaller screens
