# Analyzer Skills

Use this skill pack when comparing the implementation to the references.

## Skill Areas

- HTML parity review
  - compare structure, panels, CTA presence, and required flows to `frontend/index.html`
- screenshot parity review
  - compare spacing, section order, panel prominence, and visual density to `frontend/screenshots/`
- missing-flow detection
  - identify absent guest flow, model switching, quick actions, filters, or modal/detail paths
- capability-routing verification
  - ensure unsupported tasks become facilitation prompts instead of broken paths
- performance and cost review
  - flag large-list rendering risks, unnecessary fetches, and avoidable provider cost risks
- fix-loop enforcement
  - translate gaps into specific next actions until the output is a close match
