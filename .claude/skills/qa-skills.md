# QA Skills

Use this skill pack when validating NexusAI.

## Skill Areas

- route traversal
  - verify each primary route and key transition path
- interaction coverage
  - test buttons, chips, filters, sliders, quick actions, and auth controls
- chat validation
  - verify active model updates, message send flow, and fallback capability prompts
- guest-flow validation
  - confirm local persistence, exact expiry timing, and upgrade-to-account behavior
- regression reporting
  - keep findings reproducible, severity-ranked, and tied to impacted routes or modules
- responsive review
  - verify mobile and desktop layouts preserve key navigation and actions
