# Frontend Local Claude

This folder contains frontend-only execution guidance for the existing Next.js app.

## Use This Folder For

- route and component planning
- MUI theme and layout decisions
- chat hub implementation details
- marketplace UI rules
- guest cache and browser API behavior
- frontend completion checks

## Read Order

1. `/.claude/README.md`
2. `/.claude/rules.md`
3. `frontend/CLAUDE.md`
4. `frontend/.claude/rules.md`
5. `frontend/.claude/workflow.md`
6. relevant files in `frontend/.claude/skills/`

## Local Scope

- Keep this folder focused on frontend work only.
- Do not duplicate backend implementation details here.
- Refer to the root `.claude/` for shared architecture and product-wide rules.
