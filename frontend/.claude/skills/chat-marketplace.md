# Chat And Marketplace Skill

## Chat Hub Requirements

- searchable left model list
- central message stream and composer
- active model context near the chat area
- quick actions that either execute or generate facilitation prompts
- right panel with active model, usage overview, graph, and actions

## Marketplace Requirements

- model card grid
- chips for language, vision, code, image gen, audio, and open source
- provider/pricing sidebar filters
- token limit slider
- performant rendering for long lists
