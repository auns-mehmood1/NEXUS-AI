# UI Delivery Skill

## Priorities

- preserve the reference information hierarchy
- keep cards, filters, and layout zones modular
- make responsive behavior intentional
- avoid visual drift from the screenshots

## Implementation Patterns

- route groups for marketing, auth, and app shells
- shared layout primitives for nav, sidebars, cards, and section headers
- typed view models for model cards and right-panel metrics
- reusable chip, filter, and quick-action components
